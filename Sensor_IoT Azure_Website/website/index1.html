What is Smart Farming?

Smart farming adalah konsep pertanian yang menggunakan teknologi modern, termasuk Internet of Things (IoT), kecerdasan buatan (AI), dan analitik data, untuk meningkatkan efisiensi, kualitas, dan kuantitas produksi pertanian. Teknologi ini memungkinkan pemantauan real-time atas kondisi seperti kelembaban tanah, cuaca, dan kesehatan tanaman, serta memberikan data untuk pengambilan keputusan yang lebih tepat
Dalam praktiknya, perangkat IoT mengumpulkan data dari sensor yang ditempatkan di ladang. Data ini diproses menggunakan algoritma AI untuk mengoptimalkan irigasi, penggunaan pupuk, dan pengendalian hama, sehingga sumber daya digunakan secara efisien. Selain itu, sistem cerdas ini dapat mengurangi biaya operasional dan dampak lingkungan dengan cara yang berkelanjutan.
Menurut penelitian, adopsi smart farming dapat membantu menghadapi tantangan global seperti perubahan iklim, kelangkaan sumber daya, dan kebutuhan pangan yang meningkat seiring dengan pertumbuhan populasi dunia. Teknologi ini menciptakan solusi berbasis data untuk meningkatkan produktivitas sekaligus menjaga keberlanjutan ekosistem pertanian.

How the System Work?

ESP32, DHT22, HCSR04

Sistem IoT yang dibuat terdiri dari sensor DHT22, sensor Ultrasonic HCSR-04, ESP32, buzzer, motor servo, MQTT sebagai protokol komunikasi dan Microsoft Azure sebagai cloud. Mikrokontroler (ESP32) menginisialisasi sistem dan memastikan koneksi internet tersedia untuk memulai proses. Data dari sensor DHT22 serta sensor ultrasonik HC-SR04 dibaca setelah mikrokontroler diinisialisasi pasca koneksi berhasil. Sensor DHT22 membaca kelembaban udara; sensor ultrasonik HC-SR04 mengukur ketinggian dari tempat pakan ternak. Secara terpisah, logika kontrol diterapkan pada masing-masing sensor ini. Buzzer akan diaktifkan jika kelembaban yang terdeteksi oleh sensor DHT22 melebihi 50 persen. Motor servo akan menyala atau bergerak jika sensor ultrasonik mendeteksi ketinggian dari pakan ternak sejauh 68 cm dari sensor. Sistem segera memeriksa pengiriman data ke cloud setelah kontrol selesai. Mikrokontroler kemudian mengirimkan data ke MQTT Broker melalui protokol komunikasi MQTT. Microsoft Azure's cloud platform berkomunikasi dengan mikrokontroler melalui MQTT sebagai protokol komunikasi. Setelah data diterima oleh broker, data akan dikirimkan ke Microsoft Azure untuk disimpan serta diproses. Di dalam platform Azure, data kelembaban dari DHT22 serta mengukur ketinggian pakan ternak dari sensor ultrasonik, divisualisasikan dala bentuk grafik yang dapat digunakan untuk memantau kondisi secara real-time serta mengakses informasi secara remote (jarak jauh).
